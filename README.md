## 🛠️ Tecnologias Utilizadas

- **React.js** — Biblioteca principal
- **React Router DOM** — Gerenciamento de rotas
- **CSS Modules** — Estilização com escopo local
- **Vercel** — Deploy e hospedagem
- **JavaScript**
- **Responsividade com media queries**
